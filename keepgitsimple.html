Why Git?

  • Fully distributed

  • Every workstation is a backup

Git off-line

  • Faster

  • Commit changes
  • Merge/Rebase branches
  • Get a previous version of a file
  • Switch branches

How Git works?

  • Snapshots, not Differences
  • Integrity (sha1)
  • Only adds data
